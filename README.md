## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/Next.js-Portfolio.git
   ```
2. Change the directory
   ```bash
   cd Next.js-Portfolio
   ```
3. Install the packages
   ```bash
   npm install
   ```
4. Run the project
   ```bash
   npm run dev
   ```

## TechStack

1. Next.js
2. Typescript
3. Tailwind.css
4. Shadcdn
